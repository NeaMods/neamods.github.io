If you want to contribute, here you will find how.

All the mods are stored in GitHub repositories and this site is just a fancy link collection of those. Mod repositories are nested under the OpenModHub Github organization. This allows version control and well structured storage with free access to all.

If you would like to contribute to an exsisting mod, fork the corresponding GitHub repository and make a pull request. You will find documentation on how to do these on the GitHub website or all around the internet.

If you have an entirely new mod that you would like to add to this site, you can do so through GitHub pull requests to the (openmodhub.github.io)[https://github.com/openmodhub/openmodhub.github.io] repository that contains these pages. We will only accept GitHub repositories that contain detailed descriptions on how to make the mod.

If you would like to become a memeber of the OpenMudHub Github organization, please don't hesitate to reach out either through out contact form or directly through GitHub.
